Links to free/open source meshing software

I would like to make a real page once about these tools with more information than just listing the links.

For completeness I'll link you to the mother of all overviews of mesh generation software: Robert Schneiders' links to mesh generation software. Why create my own list if such a wonderful resource already exists? A lot of the software on Schneiders' list is old and/or not free. That is why I made my own list. NB: it is not complete. Please tell me if you think the list is lacking a vital free meshing program/project.

Reasonably recent and hopefully actively developed free meshing (related) software:

  • TetGen: A Quality Tetrahedral Mesh Generator
  • NETGEN - automatic mesh generator
  • Gmsh: a three-dimensional finite element mesh generator
  • Afront - Advancing Front (Re)Meshing Algorithm
  • MeshLab
  • QSlim - simplify triangular meshes
  • T3D Mesh Generator
  • ReMESH, SHREC and Plumber
  • JMeshLib - A C++ API to manage manifold triangle meshes
  • Discretizer, a free mesh program
  • libMesh - A C++ Finite Element Library
  • vmtk - Vascliar Modeling Toolkit
  • Generators of anisotropic adaptive grids (FORTRAN-77)

Other geometry related software:

  • CGAL - Computational Geometry Algorithms Library
  • CGM - The Common Geometry Module
  • Antiprism - a set of programs for generating, manipulating, transforming and visualising polyhedra.
  • SVAT - Scientific Visualization Artist Tools
  • stl2pov - generate a POVray mesh from an ASCII stl file
  • JavaView - Interactive 3D Geometry and Visualization JavaView
  • vxt: A class library and tools for anti-aliased voxelization of geometric objects (also: f3d and iseg)
  • openNURBS Initative

CAD software for Linux:

  • QCad (2D)
  • BRL-CAD
  • gCAD3D
  • CYCAS CAD 2D + 3D + ARCHITECTURE (commercial; free limited version)
  • VariCAD 3D/2D CAD system (commercial; trial version)

Numerical libraries:

  • FLENS: Flexible Library for Efficient Numerical Solutions
